Sorting Algorithm Visualizer

Python

Interactive app to observe the behavior of various sorting algorithms in real time, including Quick Sort, Bubble Sort, and Insertion Sort, implementing adjustable speed and dynamic block count

Game Foundation

01 Blocks Setup

Generates random list of blocks based on chosen number of blocks and min, max block heights

03 Generator

Manages state of sorting algorithms and controls the animation by yielding after each step, allowing the sorting process can be visualized in real time. Each step is processed one at a time and the display updated accordingly.

02 Control Information

Displays current state of algorithm and possible user controls for customizations

User Customization

01 Dynamic Block Count

03 Ascending or Descending

02 Adjustable Speed

04 Type of Sort

By pressing various control keys, the user can change the block count, speed, whether the sorting is in ascending or descending, and the type of sort. These customizations allow the user to understand and visualize each algorithm easier. The changes are displayed in real time within the top labels.

Implementing Algorithms

Bubble Sort

Repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order

Insertion Sort

Builds final sorted array one item at a time by inserting each new element into its correct position within the sorted portion

Quick Sort

Recursively partitions the array into smaller sub-arrays around a pivot and then sorts the sub-arrays
