Weather App

JavaScript, API, UI/UX Design

Interactive app where users can input a city name and discover the current temperature, weather condition, and country

Basic HTML Structure

01 Search Bar

Allows users to type and input city name and creates a location card based on input

03 Location Card

Each Location card includes an icon representing weather condition, the temperature in Celsius, and city, country name

02 Notification Banner

If user inputs invalid city name or repeated city name, error will appear below search bar

Designing UI/UX with CSS

01 Color Palette

03 Grid Structure

02 Icons

04 Responsive Layout

To make the app cohesive, I chose a green, brown, and white color pallette as well as used a weather icon set. Using a grid structure, the location cards easily appear next to each other when added. Lastly, to make the app scalable without empty space on all screens, I changed the card information to appear vertical on larger screens and horizonal on smaller screens.

Backend Development with JavaScript

Weather API

Calls on Open Weather API for real time temperature of cities

Repeated Inputs or Errors

Uses Arraylist to store displayed cards, prevent repeats, and trigger notification banner

Two Input Types

Allows user to use CITY, COUNTRY OR CITY input which is useful with city names that are identical in different countries
